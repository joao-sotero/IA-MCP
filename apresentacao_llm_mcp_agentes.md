# 🚀 LLM, Agentes, MCP e IA
## Entenda como essas tecnologias se conectam

---

# 🧠 O que é IA?

**IA (Inteligência Artificial)** é o conceito geral.

Inclui:
- Machine Learning
- Deep Learning
- LLMs
- Visão computacional

💡 **Resumo:**
> IA = qualquer sistema que simula inteligência humana

---

# 🤖 O que é um LLM?

**LLM (Large Language Model)**

Capaz de:
- Entender linguagem natural
- Gerar texto
- Responder perguntas

✅ Exemplo de uso:
- Chatbots
- Assistentes
- Geração de conteúdo

⚠️ Limitações:
- Não executa ações sozinho
- Não acessa sistemas diretamente

---

# 🔌 O que é MCP?

**MCP (Model Context Protocol)**

👉 É um padrão que conecta LLMs a ferramentas externas

Permite acesso a:
- APIs
- Bancos de dados
- Sistemas internos
- Serviços (Docker, etc)

💡 **Resumo:**
> MCP = ponte entre IA e o mundo real

---

# 🤖 O que são Agentes?

**Agente = LLM + ferramentas + tomada de decisão**

Capaz de:
- Planejar ações
- Executar tarefas
- Usar ferramentas automaticamente
- Resolver problemas em múltiplos passos

---

# 🔥 Exemplo prático

### Pedido:
> "Buscar clientes inativos e enviar mensagem"

### LLM:
- Apenas responde com texto

### Agente:
1. Consulta banco de dados
2. Filtra clientes
3. Chama API de envio
4. Executa ação

---

# ⚙️ Diferença entre os conceitos

| Conceito | Função | Executa ação? |
|----------|--------|--------------|
| IA | Conceito geral | Depende |
| LLM | Processa linguagem | ❌ |
| MCP | Conecta ferramentas | ⚠️ |
| Agente | Executa tarefas | ✅ |

---

# 🧩 Arquitetura moderna

```
Usuário
  ↓
LLM (entende)
  ↓
Agente (decide)
  ↓
MCP (conecta)
  ↓
Ferramentas (API, DB, Docker)
```

---

# 🐳 Onde entra o Docker?

Docker é usado para:
- Isolar ferramentas
- Rodar MCP servers
- Padronizar ambiente

💡 Cada tool pode rodar como container

---

# 🚀 Aplicação real (exemplo)

Sistema de atendimento:

- Cliente envia mensagem
- Agente interpreta intenção
- Consulta CRM
- Responde automaticamente
- Abre ticket se necessário
